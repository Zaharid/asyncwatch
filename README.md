Asyncwatch
=========

An asyncronous
[inotify](http://man7.org/linux/man-pages/man7/inotify.7.html) wrapper based on curio.


Installing
-----------

````
pip install asyncwatch
````

Requirements
-----------

- A recent enough version of the Linux Kernel (at least 2.7).
- Python >= 3.5
- [Curio](https://github.com/dabeaz/curio)
